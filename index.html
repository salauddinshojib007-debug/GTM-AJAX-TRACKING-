How this AJAX form works

When you hit Submit, the form sends a POST request to /api/submit via jQuery's $.ajax() — without reloading the page.

The Bounteous jQuery AJAX listener intercepts the ajaxComplete event and pushes all request/response metadata into the GTM dataLayer.

GTM Setup Checklist

  1. dataLayer init — window.dataLayer = window.dataLayer || [] at the top of <head>.
  2. jQuery on the page — the AJAX listener requires it.
  3. Custom HTML tag — paste the Bounteous AJAX listener, fire on All Pages.
  4. Data Layer Variable — name: attributes.response.data.message
  5. Custom Event trigger — event name ajaxComplete, condition: DLV contains "Thanks for contacting us!"
  6. GA4 Event tag — event name generate_lead, attach the trigger above.
  7. Preview & test — submit this form, verify in GTM Debug & GA4 DebugView.
